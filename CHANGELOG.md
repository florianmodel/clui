# Changelog

## Unreleased

### Added

- Extended Simple mode Finder with first-class file support: users can now choose single files, see recent files, inspect file context in plain language, edit safe Finder-style details, and open or install matching tools directly from the file screen.
- Added friendly file metadata editing for macOS in Finder mode, including tags, locked state, and per-file extension visibility.
- Added read-only technical metadata panels for Spotlight details and extended attributes, plus guided-tool launch handoff that can prefill a selected file when a tool has one obvious file input.
